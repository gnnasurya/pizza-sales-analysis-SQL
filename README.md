# Pizza Sales Data Analysis Using SQL

## Objective
Analyze revenue performance, customer ordering behavior, and product contribution using SQL.

## Tools Used
- MySQL Workbench
- SQL

## Key Analysis Performed
- Total Revenue Calculation
- Top-Selling Products
- Category-wise Revenue Contribution
- Peak Order Time Analysis
- Top 3 Pizzas per Category (Window Functions)

## Business Insights
- Identified top 20% products contributing majority revenue
- Determined peak demand hours for staffing optimization
- Segmented revenue by product category

## SQL Concepts Used
- INNER JOIN
- GROUP BY
- Aggregate Functions (SUM, COUNT, AVG)
- Window Functions (RANK, PARTITION BY)
